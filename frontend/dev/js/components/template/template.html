Web Application Template

base model

A minimalistic approach to starting off each new project with an application template while including all the most loved components already completely configured, minus any nasty bloat.

Like most application developers, starting a new project usually means duplicating a favorite project and then taking out all the content, logic and bloat that isn't needed for the new project. Most of the time, additional components are needed for the new project and well, the reality check hits and the realization of configurations upon alterations need application to what was thought to be a "template" for new projects; hence this project initiative.

Review the readme.md file in the project's root directory for more information on setup and usage.
